script.js — New Functions

JavaScript Added

A Stat Counter Animation

Uses IntersectionObserver (threshold: 0.5) to detect when stat numbers enter the viewport, then fires a smooth count-up animation from 0 to the target value over ~1800ms at 60fps.

// Fires when .stat-number scrolls into view function animateCounter(el) { const target = parseInt(el.getAttribute('data-target'), 10); const step = Math.ceil(target / (1800 / 16)); let current = 0; const timer = setInterval(() => { current += step; if (current >= target) { current = target; clearInterval(timer); } el.textContent = current.toLocaleString(); }, 16); }

B Auction Countdown Timers

Reads data-end attribute (ISO date string) from each .auction-card__timer element and calculates a live countdown, updating every second via setInterval.

// Updates every .auction-card__timer element every 1 second function getCountdownString(endDateStr) { const diff = new Date(endDateStr) - new Date(); if (diff <= 0) return 'Ended'; const d = Math.floor(diff / 86400000); return d > 0 ? `${d}d ${h}h ${m}m` : `${h}h ${m}m ${s}s`; } setInterval(updateTimers, 1000);

C FAQ Accordion Toggle

Global function (called inline via onclick="toggleFaq(this)") that closes all open FAQ items, then opens the target item if it was previously closed. Creates an exclusive single-open accordion.

function toggleFaq(btn) { const item = btn.closest('.faq-item'); const isOpen = item.classList.contains('open'); document.querySelectorAll('.faq-item.open') .forEach(i => i.classList.remove('open')); if (!isOpen) item.classList.add('open'); }
